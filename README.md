# Prévision des ventes – Machine Learning

## Objectif
Développer un modèle de prévision des ventes futures à partir de données historiques afin d’aider les entreprises à prendre de meilleures décisions commerciales.

## Données
Données de ventes quotidiennes contenant :
- Date
- Ventes

## Méthodologie
- Nettoyage des données
- Ingénierie des caractéristiques temporelles
- Modélisation avec Random Forest
- Évaluation avec MAE et RMSE
- Visualisation des résultats

## Résultats
Le modèle prédit efficacement les tendances futures des ventes avec une faible marge d’erreur, permettant une meilleure planification stratégique.

## Technologies utilisées
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook
